# Turn-Based RPG 2D

Turn-based RPG sistem prototipe yang dibuat menggunakan unity
Prototype turn-based RPG battle system built with Unity.

## Features
- Turn-based battle
- Pilih Target
- Spawn Musuh
- State Menang dan Kalah
- Sistem Cutscene dan Dialog

## Engine
- Unity 6

## Controls
- Mouse Click : Select target
- Space : Interaksi dengan NPC

## Notes
Proyek ini dibuat sebagai bahan teknikal untuk Tes Pre-Interview
